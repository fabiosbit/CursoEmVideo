Principais Formatações:

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B (não semântica).

Nesta frase tenho um termo em destaque usando a tab STRONG (semântica)

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag I (não semântica)

Nesta frase temos um termo em ênfase usando a tag EM (semântica)

Texto Marcado:

Podemos criar um texto marcado usando a tag MARK

E no outro parágrafo temos outro texto marcado em outra cor, utilizando STYLE.

Texto Grande e Pequeno:

Texto grande a tab BIG não deve ser utilizado, para texto grande use o CSS

Para texto pequeno pode utilizar a tab SMALL para quando se referir às letras miúdas de um contrato por exemplo.

Texto Deletado e Inserido:

Nesta frase temos uma parte do texto que foi riscada e deve ser ignorada, usando a tag DEL.

Nesta frase temos um texto que foi inserido, ficando sublinhado usando a tab INS

Existe tambem a tab U para texto sublinhado, mas é (não semântica)

Texto sobrescrito ou subscrito:

Para coisas do tipo x20+10, use a tag SUP.

E para coisas do tipo H2O, use a tag SUB.

Formatações de código fonte e Pré formatação

O comando document.getElementById('teste') é escrito em JavaScript. A tag (code) deixa a fonte monoespaçada.

Abaixo temos um código utilizando a tag (code) mais a tag (pre), que leva a formação do código para o texto:

        
        num = int(input('Digite um numero'))
        if num % 2 == 0:
            print(f'O número {num} é PAR')
        else:
            print(f'O número {num} é IMPAR')
