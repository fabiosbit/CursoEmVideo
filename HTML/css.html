Trabalhando com Fontes

Para tamanhos de fontes, utilize sempre as opções 'px' ou 'em', para páginas que serão exibidas em diversos dispositivos, utilize sempre 'em', que é uma porcentagem do tamanho padrão, 1em normalmente é igual a 16px.

Escolha sempre uma família de fontes ex: "font-family: Arial, Helvetica, sans-serif;", assim o navegador tenta a primeira fonte (arial), se não tiver ele tenta (Helvica) e se ainda não tiver ele define uma automaticamente, podendo ser serifada ou não serifada (serifada possui aquela perninha nas letras)

É possível declarar todas as propriedades da fonte em uma única linha, seguindo a seguinte ordem: "font: font-style, font-weight, font-size, font-family

Fontes Externas

É sempre possível baixar novas fontes, mas também pode importar direto do google fonts, acesse o site fonts.google.com/ e copie o código CSS em @import.

Existe um plugin para o Chrome chamado 'Fonts Ninja', com ele é possível descobrir quais fontes estão sendo utilizadas em um site.

Trabalhando com posicionamento em CSS

Página com os exemplos: Posições

Propriedade Relative e Static:

O position static é o comportamento padrão de uma div, o elemento fica estático na posição inicial e não se move. Já o position relative é a posição relativa a posição original de onde o elemento foi criado, a partir desta posição inicial, vale as propriedades top, right, left e botton, que dão a quantidade de espaço desejada em cada direção e também o z-index: 0 (padrão), que posiciona o elemento no eixo z, colocando ele na frente ou atrás de outros elementos..

Propriedade Display inline

Exibe todos os elementos na mesma linha e não permite as alterações de tamanho heigh e widht e também adiciona um espaço entre cada elemento. O display: inline dá o comportamento de uma palavra para cada elemento da linha, é como fosse tudo uma linha de texto.

O comando text-align irá alinhar somente o texto e não o elemento inteiro, o elemento div, h1, p, etc, irá continuar ocupando o mesmo espaço e posição.

Propriedade Display block

Block é a propriedade padrão dos elementos, ele permite definir heigh e widht e ocupa toda a linha onde ele é inserido, sendo possível apenas colocar um block debaixo do outro.

Utilize a propriedade margin-left: auto para colcoar o elemento na esquerda, centro ou direita da página, a opção auto preenche toda a margem a direita ou esquerda, des de que uma largura tenha sido definida para o elemento.

Propriedade Display inline-block

O inline-block junta as propriedades inline e block ao mesmo tempo, os elementos se comportarão como palavra e bloco, o text-align pode ser usado para alinhar o texto e/ou o elemento inteiro.

Para usar o alinhamento justificado, é preciso adicionar uma linha a mais, pois a última linha nunca é justificada, e se só tiver uma linha, ela mesma é a última.

Propriedade Float

Quando é aplicado float em um elemento, ele ganha um novo layer, deixando um "buraco" no layer inicial, isso faz com que o conteúdo do layer 0 suba e ocupa o lugar do elemento que flutuou. O elemento pai que contem um elemento float, perde sua altura e largura, isso pode ser recuperado usando o overflow.

O conteúdo que está dentro do elemento float nunca é escondido, se tiver um texto dentro de uma div que é float, esse texto será colocado logo abaixo ou ao lado do elemento, fazendo com que o texto sempre apareça na página. Isso quando algum outro elemento ocupar o espaço que o float deixou no layer 0. Os elementos se sobrepõe, mas o conteúdo não.

Se eu quiser que o navegador considere os espaços deixados pelos floats no layer 0 e mantenha esses espaços ocupados para que o resto dos elementos não subam e ocupem esses espaços, posso utilizar a propriedade overflow: hidden; no elemento pai.

Se eu quiser que um conteúdo não ocupe o local de um elemento float, existe a propriedade clear: both; isso faz com que o conteúdo verifique se existe algum elemento flutuado a direita ou a esquerda, se tiver ele sai desse espaço e desce.

Propriedade position absolute

A posição padrão dos elementos é static, que não aceita alterações de posição.

O position absolute funciona da mesma forma que o float, ele cria um novo layer para aquele elemento, mas a referencia de posição é o tamanho da tela do navegador.

A propriedade de posicionamento left, right, top e botton, tem seus valores referente ao tamanho da tela naquele momento, ou seja, se o navegador está com largura de 1000px, se eu colocar right: 50%; a borda de início do elemento irá se deslocar 500px.

Para centralizar um elemento na tela, devo utilizar a propriedade right ou left 50% junto com a propriedade transform: translateX(-ou+50%) , isso move 50% da largura do elemento em vez da largura da tela.

Um elemento filho absolut só irá respeitar o tamanho do elemento pai, se o pai não for static.
